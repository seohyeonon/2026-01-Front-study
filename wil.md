# 오늘 공부한 내용
숫자를 코드로 바꾸고, 숫자가 바뀌는 행위를 버튼으로 핸들링하여 특정 id의 텍스트를 코드로 핸들링하는 법을 배웠습니다.
script 태그의 위치는 body의 맨 아래에 위치합니다. 브라우저는 html 문서를 한 줄 한 줄 순차적으로 읽기(파싱)을 하는데 html 파싱 중에 script를 발견하게 된다면 네가지 상황이 발생합니다. 
1. 파싱 중단
2. script 파일을 다운 받는다
3. script 파일 실행
4. html을 이어서 파싱
이럴 경우 생기는 문제는 화면이 늦게 뜨고 순서가 꼬일 수 있다는 점입니다.
따라서 script 태그는 body의 맨 아래에 위치해야 정상적으로 작동할 수 있습니다.
